# 💰 Personal Expense Tracker for Financial Management

A **machine learning powered web application** that helps users track income, manage expenses, visualize spending patterns, and forecast future financial trends.

Built using **Streamlit, Python, and Machine Learning algorithms**, this project provides interactive dashboards and predictive insights for smarter financial decision-making.

---

# 📌 Project Overview

Managing personal finances manually can be difficult and time-consuming. The **Personal Expense Tracker** simplifies financial management by allowing users to:

* Record **daily income and expenses**
* Visualize spending patterns through **interactive charts**
* Analyze financial data with **real-time dashboards**
* Predict future expenses using **Machine Learning models**

The application integrates **data analytics and forecasting techniques** to help users understand their financial habits and make informed decisions.

---

# 📄 Abstract

This project presents the design and development of a **real-time Expense Tracker web application with integrated machine learning–based financial forecasting**.

Developed using **Streamlit**, along with Python libraries such as **pandas, scikit-learn, and plotly**, the system enables users to log their income and expenses, visualize spending patterns, and receive predictive insights.

Forecasting is achieved using algorithms like **Support Vector Regression (SVR)** and **Random Forest**, providing personalized expense predictions based on historical data.

The platform features **interactive dashboards, dynamic charts, and intuitive navigation** powered by tools like **streamlit-option-menu and plotly express**, offering a seamless and insightful user experience.

By combining **financial tracking with predictive analytics**, the application helps users make informed decisions and better manage their personal finances.

# 🏗️ System Architecture

Below is the architecture workflow of the system.

![System Architecture](images/expense-tracker-methodology.jpeg)

### Workflow Explanation

1. **User Registration**
   Users create an account to securely access the system.

2. **User Dashboard**
   Displays an overview of financial information.

3. **Expense Entry Form**
   Users enter income and expense transactions.

4. **Machine Learning Training**
   Historical data is processed to train ML models.

5. **SQLite Database Storage**
   All user financial records are stored in the database.

6. **Forecasting Module**
   Time series machine learning models predict future expenses.

7. **Visualization and Insights**
   Charts and reports help users analyze financial patterns.

# 🚀 How to Run the Project

### Step 1: Clone the Repository

```bash
git clone https://github.com/yourusername/personal-expense-tracker.git
```

---

### Step 2: Navigate to the Project Folder

```bash
cd personal-expense-tracker
```

---

### Step 3: Install Required Libraries

```bash
pip install -r requirements.txt
```

---

### Step 4: Run the Application

```bash
streamlit run app.py
```

The application will start at:

```
http://localhost:8501
```
